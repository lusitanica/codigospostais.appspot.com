/cp4?

Devolve as coordenadas geográficas (latitude e longitude) para um determinado código postal de 4 dígitos.

Aceita pedidos GET.

Parâmetro: codigo código postal de 4 dígitos.

Parâmetro opcional: callback para a chamada de uma função em javascript (JSONP).

Resposta: JSON

http://codigospostais.appspot.com/cp4?codigo=2675

/cp7?

Devolve, quando existem, a Localidade, Artéria, Local/Zona e Troço para um determinado código postal de 7 dígitos.

Aceita pedidos GET.

Parâmetro: codigo código postal de 7 dígitos.

Parâmetro opcional: callback para a chamada de uma função em javascript (JSONP).

Resposta: JSON

http://codigospostais.appspot.com/cp7?codigo=2620275

/coord2cp7?

Devolve, quando existe, o código postal de 7 dígitos para um determinado par de coordenadas geográficas (latitude e longitude).

Aceita pedidos GET.

Parâmetro: key chave do cliente.

Parâmetro: lat latitude WGS84.

Parâmetro: lon longitude WGS84.

Parâmetro opcional: callback para a chamada de uma função em javascript (JSONP).

Resposta: JSON

http://codigospostais.appspot.com/coord2cp7_username?
